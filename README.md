## Starter Code For Nodejs + Typescript + Nodejs + Winston + Prettier +Mongoose

```
Node Version : 22.14.0
Package Manager : Pnpm
```

## To use the mention nodejs version (Nvm must be installed)

```
nvm use
```

## Getting Started

```
pnpm install
pnpm run dev
```

## Format Code

```
pnpm format
```

## Build Code

```
pnpm build
pnpm start
```
